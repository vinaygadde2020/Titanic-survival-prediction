# Titanic Survival Prediction 🚢

## 📌 Project Overview
This project uses Machine Learning to predict whether a passenger survived the Titanic disaster based on passenger data such as age, gender, ticket class, and more.

## 🎯 Objective
To build a classification model that predicts survival of Titanic passengers.

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Google Colab

## 📊 Workflow
- Data Collection
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Model Building
- Model Evaluation

## 📈 Result
Achieved good accuracy using machine learning classification models.

## 🚀 Future Improvements
- Try advanced models like Random Forest and XGBoost
- Improve feature engineering
- Hyperparameter tuning
